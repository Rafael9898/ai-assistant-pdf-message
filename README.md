# AI Assistant (PDF or Message)

This is a modern, responsive full-stack chatbot built with React and Next.js. It demonstrates how to create a custom user interface from scratch and integrate it with powerful AI models using the RAG (Retrieval-Augmented Generation) concept to read and understand specific contexts.

# Features

* **Custom Chat Interface:** Built from scratch with React components.

* **AI Integration:** Seamless connection to an AI API (Google Gemini) via Next.js serverless routes, keeping API keys secure.

* **Context-Aware (RAG):** The bot can answer questions based on a specific context provided (such as text or a PDF), instead of just giving simple answers.

* **Responsive Design:** Optimized for viewing on desktops and mobile devices.

# Tech Stack

* **Frontend:** React, Next.js, TailwindCSS
* **Backend:** Next.js API Routes 
* **AI:** Google Gemini API
* **Deployment:** Vercel
